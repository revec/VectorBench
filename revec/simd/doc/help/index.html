Simd Library Documentation

This is the reference documentation of Simd Library, the C++ image processing library. This documentation have been generated using the tool doxygen. It contains a detailed description of all classes and functions of the Simd Library. If you have downloaded the Simd Library package, you actually have a local copy of these pages in the simd/doc/help/ directory.

Use the menu above to navigate through the documentation pages. As a first step, you may look at the list of available modules.
